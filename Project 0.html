The internet

The internet consists of computers that can communicate through html documents. A server receives a request and sends back documents in the form of HTML.

The request that is send by the browser is called: >HTTP Request
The documents that a browser can receive from the server are called HTML documents

HTML Tags

Tags examples: a, b, br, em, h1 and title

HTML tags are being used in this document to differentiate the text from each other with markup or behavior

HTML = HyperText Markup Language
Tags can contain attributes so that certain sources can be called or linked to. Examples of attributes are: "src" and "href" Also important to know is that there are inline and block elements that behave diferently in a browser.
